# Skill: Map and Audit Game Systems & Resource Flows
#game-balance #system-design

## Kapan Harus Menggunakan
Gunakan skill ini ketika merancang arsitektur ekonomi game, memetakan hubungan antar subsystem, menentukan tipe resource, atau mendiagnosis kebocoran dan inflasi dalam game agar struktur mekanik tidak menjadi "tumpukan spageti".

## Deskripsi
Fokus pada perancangan sistem inti ekonomi game melalui pemetaan objek dan aliran sumber daya menggunakan analogi keran dan wadah air.

## Alur Kerja (Workflow)
1. **Inventory Objects & Resources:** Daftarkan semua entitas dan elemen yang mengalir di dalam game.
2. **Define Subsystem Relations:** Hubungkan subsystem lokal dengan master system utama.
3. **Chart Flow Anatomy:** Petakan sirkulasi berdasarkan Keran (Inflow), Wadah (Pool), dan Pembuangan (Sink/Outflow).
4. **Diagnose Economy Health:** Periksa potensi terjadinya inflasi besar atau kelangkaan kritis sebelum masuk tahap produksi.

## 🔗 Hubungan Keterkaitan
- [[Apply Balance Foundations]] — Menyediakan cetak biru visual untuk ekosistem roda gigi game.
- [[Establish Math Anchors]] — Memerlukan penentuan angka dasar sebelum aliran resource dapat disimulasikan.
