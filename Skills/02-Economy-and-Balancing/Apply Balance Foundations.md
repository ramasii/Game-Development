# Skill: Apply Game Balance Foundations (Gear Machine Framework)
#game-balance #framework

## Kapan Harus Menggunakan
Gunakan skill ini ketika mendesain, menganalisis, atau mengevaluasi keseimbangan mekanik, ekonomi, dan tingkat kesulitan sebuah game agar semua elemen saling terikat secara harmonis.

## Deskripsi
Mengadopsi kerangka kerja *Gear Machine*—di mana setiap elemen dalam game saling terikat layaknya roda gigi; mengubah satu nilai/bagian akan memengaruhi seluruh ekosistem game.

## Alur Kerja (Workflow)
1. **Define Objective & Audience:** Tentukan tipe monetisasi (F2P/Paid) dan target psikologi kesulitan game.
2. **Establish Core Math:** Susun draf biaya (Cost) vs dampak (Utility) awal di spreadsheet.
3. **Identify Connected Gears:** Analisis efek domino pada komponen lain jika nilai variabel X diubah.
4. **Formulate Playtest Goals:** Tentukan metrik subjektif untuk divalidasi guna menghindari bias desainer (*Designer Madness*).
