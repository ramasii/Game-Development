# Skill: Identify and Design Core Gameplay Loops
#game-design #loop

## Kapan Harus Menggunakan
Gunakan skill ini di awal proses pengembangan sebelum membuat class atau menulis kode program, untuk mengidentifikasi ritme dan struktur aksi berulang yang akan dialami oleh pemain sepanjang sesi permainan - Core Loop.pdf].

## Deskripsi
Menentukan urutan siklus tindakan pemain dan respons balik dari sistem yang menjadi tulang punggung utama dari seluruh pengalaman bermain game - Core Loop.pdf].

## Alur Kerja (Workflow)
1. **Isolate Primary Action:** Temukan satu tindakan utama yang selalu diulang pemain setiap ronde - Core Loop.pdf].
2. **Document Sequential Flow:** Catat alur mulai dari aksi, kalkulasi skor, hingga pemberian reward - Core Loop.pdf].
3. **Build Prototype Dummy:** Implementasikan `GameManager` sederhana untuk menguji kelancaran loop sebelum menambah fitur kompleks - Core Loop.pdf].
