# 🧭 Skills — Second Brain Index

Seluruh pustaka pengetahuan dibagi menjadi beberapa **ranah keahlian (kategori)**. Setiap kategori punya folder dan file index-nya sendiri.

---
## ➕ Cara Menambahkan Skill Baru

> [!tip] Panduan Kontribusi Skill
> 1. **Tentukan kategori** yang paling cocok dari daftar di atas.
> 2. **Buat file `.md` baru** di dalam folder kategori tersebut, dengan nama dan deskripsi yang jelas.
> 3. **Patuhi aturan** pada [[Standar Format File Skill]] agar catatan rapi dan mudah dibaca AI.
> 4. **Daftarkan skill** ke file index kategorinya (misal: `🎯 1. System Deconstruction.md`) dengan format:
>    `- [[Nama Skill]] — Deskripsi singkat skill ini.`
> 5. Jika **tidak ada kategori yang cocok**, buat folder baru dengan format:
>    `XX-Nama-Kategori/` lalu buat file index-nya dan daftarkan di halaman ini.
