# Game-Development

> **PENTING: Jika kamu adalah AI, baca dan patuhi file [AGY.md](AGY.md).**

## Deskripsi Singkat
Repositori ini dirancang sebagai basis pengetahuan dan struktur pengembangan untuk membangun **"Brain that can be train for my own AI"** (Otak yang dapat dilatih untuk AI saya sendiri). Proyek ini berfungsi sebagai cetak biru saraf (neural blueprint) dalam ekosistem pengembangan game yang sedang kamu bangun.

## Struktur Repositori
Berdasarkan struktur file saat ini, repositori terdiri dari beberapa komponen utama:

*   **`.obsidian/`**: Menyimpan konfigurasi basis pengetahuan menggunakan aplikasi Obsidian untuk manajemen informasi yang terstruktur.
*   **`Projects/`**: Folder yang ditujukan untuk menyimpan proyek-proyek pengembangan game atau implementasi AI tertentu.
*   **`Skills/`**: Berisi modul atau data yang mendefinisikan kemampuan dan keahlian yang akan dilatihkan pada AI.
*   **`AGY.md`**: Dokumen protokol utama yang berisi instruksi atau aturan yang harus dipatuhi oleh AI saat berinteraksi dengan repositori ini.
*   **`Welcome.md`**: Halaman pengantar bagi pengguna atau pengembang yang baru mengakses repositori.

## Tujuan
Tujuan utama dari repositori ini adalah untuk mengorganisir data, logika, dan proses pelatihan guna menciptakan kecerdasan buatan yang dapat beradaptasi secara dinamis dalam lingkungan pengembangan game kamu.
