# Profil Developer: Rama (AGY)

## 👤 Tentang Saya
- **Nama**: Rama
- **Peran Utama**: Unity Programmer
- **Sub-peran**: 3D Artist (Blender), Sound Engineer (FL Studio)
- **Fokus Tim**: Ideasi, Game Design, Prototyping
- **Status**: Mahasiswa Teknologi Game (Tahun ke-2)

## 🎯 Target & Tujuan Tahun Ini
- **Kuliah**: Merangkum dan menyelesaikan tugas game development secara efisien.
- **Pengembangan Skill**:
  - Mempelajari *game design* agar game lebih menarik dan *engaging*.
  - Mempelajari *design pattern* agar struktur kode rapi, *maintainable*, dan *scalable*.
  - Merangkum materi belajar secara teratur untuk memperkuat "otak kedua".
- **Implementasi**: Membuat game baru dengan menerapkan ilmu/materi yang telah dipelajari.

## 🗣️ Preferensi Komunikasi
- **Gaya Bahasa**: Santai, kasual layaknya teman sendiri, tapi pengen dipanggil "King".
- **Format Penjelasan**: Langsung *to the point* (ringkas & padat), tetapi siap menjelaskan secara mendalam jika diminta detailnya.

## 💪 Kekuatan & Kelemahan
### Kekuatan (Strengths)
- Rasa ingin tahu (*curiosity*) yang tinggi.
- Pengalaman menggunakan Unity yang cukup lama.
- Kemampuan menggunakan Blender 3D yang mumpuni.
- Cepat dalam membuat prototype (*rapid prototyping*).

### Kelemahan (Weaknesses)
- Terkadang bingung membuat game terasa menyenangkan/seru (*game feel* & *engagement*).
- Kesulitan menemukan/mengembangkan ide game yang pas.
- Tantangan dalam berkomunikasi dan berkolaborasi dalam tim.
- Kecenderungan untuk mengerjakan segala hal sendirian (*lone wolf*).
- Sebagai programmer, kesulitan menentukan *art style* yang konsisten untuk game.
- Kesulitan menentukan desain UI dan UX yang baik.
